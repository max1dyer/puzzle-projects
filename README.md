# Puzzle Projects
My versions of popular logic games with each folder corresponding to a puzzle remake.

## Wordle Terminal
A terminal-based version of Wordle built in Python with color-coded output to indicate letter positioning in guesses.

## Wordle GUI
A desktop version of the popular Wordle game built in Python using Tkinter. This project recreates the Wordle gameplay experience with an interactive GUI, animated feedback, and both physical/on-screen keyboard support.
